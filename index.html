Welcome to Corazon Heart

An open source C# library intended to make your development life easier. From customizing your website language, cryptography, obfuscation and compression, it's in the package. It is still under heavy development though. To clone the Git repository, run the following commands:

$ cd kendtimothy/corazon-heart
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
